# Brainfuck-rust

## About 🤓

brainfuck-rust is a Brainfuck interpreter written in Rust and based on [brainfuck-c](https://github.com/kgabis/brainfuck-c)

## Usage 🛠️

you compile this code

## Examples 🤔

```bash
./brainfuck.exe ./hw.bf
```
